# Lunte

A self-contained JavaScript linter, mostly matching non-formatting rules of Standard.

## Packages

- [`lunte`](packages/lunte/README.md) — CLI and core analysis engine
- [`lunte-lsp`](packages/lunte-lsp/README.md) — stdio Language Server Protocol wrapper
- [`vscode-lunte`](packages/vscode-lunte/README.md) — minimal VS Code client for `lunte-lsp`

## Quick start

```sh
npm install --save-dev lunte
```

```sh
lunte [optional: dir or glob]
```
